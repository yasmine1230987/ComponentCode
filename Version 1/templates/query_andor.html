Query the Database - AND/OR

Query and/or is used to sort the data simultaneously, one can find parts that are equal to a certain materialID but also have more than a certain type of code. You can sort based off material ID, batchID, or CodeA,B,C,D. You are also able to use AND as in both your conditions must be met, or OR where either of your conditions may be met. Subsequent data is outputted when submitted in the same format as "show table" with rowID as the first value


For Column 1 insert the name of the first subject you'd like to sort by, for example "CodeB"


For operator 1, insert a symbol like >, <, >=, <=, or =


Enter a numeric value for comparison used with Column 1 and Operator 1


For Column 2 insert the name of the second subject you'd like to sort by, for example "Material"


For operator 2, insert a symbol like >, <, >=, <=, or =


Enter a numeric value for comparison used with Column 2 and Operator 2
